Projects

Brief descriptions of some of the projects I've worked on. Because of legal restrictions, I am not able to share comprehensive examples of work that was done for Achievement First.

Data Heroes Training - Ongoing

I'm currently working with a colleague to pilot a training program designed to teach selected Achievement First employees how to visualize and analyze data using Tableau and our SchoolZilla data warehouse. The program consists of three 4-hour hands-on sessions, several homework assignments and a culminating project.

SAM Reporting

As part of Achievement First's ground-breaking teacher development and evaluation system, Teacher Career Pathway, I designed, created and managed the deployment of a number of reports surfacing student achievement and teacher impact on student growth. The reports were built using Tableau with a SQL Server back-end.
